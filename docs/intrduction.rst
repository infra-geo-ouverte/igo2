Sommaire
=======

|igo2| est en développement afin de fournir un navigateur géographique qui supporte les standards OGC et qui est basé sur |openlayers|, |ang2| et |bootstr|.  A typical use case met by the plugin involves one or more users checking out a local working copy to edit features in (potentially) offline mode using SpatiaLite and committing changes back to the PostGIS server.

Plus d'information est disponible à la page `original page`_.
